# IW-GRUPO5-E2
Nombre del proyecto: Deusto Plant Operations S.L.
Estructura del proyecto: Una aplicación llamada Gestión
El proyecto corresponde al Reto 4 y permite gestionar turnos de trabajo y partes de trabajo diarios del personal de planta de Deusto Plant Operations S.L.
Integrantes del grupo: Diego Suarez, Juan Maria Díaz y David Orive.
1.- Enlace al repositorio:
https://github.com/Txemaaa/IW-GRUPO5-E2.git
2.- Funcionalidades adicionales implementadas
No se han implementado funcionalidades adicionales más allá de las mínimas requeridas por el reto.
Las funcionalidades mínimas implementadas son:
Gestión de turnos: alta, listado, detalle, actualización y baja.
Gestión de partes de trabajo: alta, listado, detalle, actualización y baja.
Gestión de empleados: alta, listado, detalle, actualización y baja.
Migrar la base de datos a local para tener el archivo db.sqlite3 como requisito de la E2.
